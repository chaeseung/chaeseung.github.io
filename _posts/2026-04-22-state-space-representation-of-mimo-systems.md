동기 발전기와 같이 입력과 출력이 여러 개인 **MIMO(Multi-Input Multi-Output, 다변수)** 시스템을 모델링할 때 가장 적합한 방법은 **상태공간방정식**이다.

## 1. 상태공간방정식(State Space Equation)의 특징
전달함수 방식이 시스템의 입력과 출력 사이의 관계(외부 특성)에만 집중하는 것과 달리, 상태공간법은 시스템의 내부 상태 변수를 포함하여 모델링한다.

* **행렬 기반 표현**: 시스템의 상태($x$), 입력($u$), 출력($y$)을 행렬로 표현하여 다변수 데이터를 체계적으로 처리한다.
* **표준 형식**:
    $$\dot{x}(t) = Ax(t) + Bu(t) \quad \text{(상태 방정식)}$$
    $$y(t) = Cx(t) + Du(t) \quad \text{(출력 방정식)}$$
* **현대 제어 이론**: 고전 제어 이론(전달함수 중심)의 한계를 극복하고, 복잡한 비선형 시스템이나 시변 시스템, MIMO 시스템을 해석하는 현대 제어의 기초가 된다.



## 2. 모델링 도구별 비교

| 구분 | 주요 용도 및 특징 | 한계점 |
| :--- | :--- | :--- |
| **상태공간방정식** | **MIMO 시스템**, 내부 상태 파악, 현대 제어 | 행렬 연산의 복잡성 |
| **라플라스 변환** | **SISO 시스템**, 주파수 영역 해석, 전달함수 도출 | 다변수 시스템 표현의 복잡성 |
| **푸리에 급수** | 비정현 주기 신호의 성분 분석 | 시스템의 동적 모델링에는 부적합 |
| **z-변환** | **디지털(이산 시간) 시스템**의 해석 및 설계 | 연속 시간 시스템 해석에는 부적합 |

## 3. 핵심 요약
* **SISO(Single-Input Single-Output)**: 입력 1개, 출력 1개 $\rightarrow$ **전달함수(라플라스 변환)**가 유리.
* **MIMO(Multi-Input Multi-Output)**: 입력 다수, 출력 다수 $\rightarrow$ **상태공간방정식**이 필수적.
* 상태공간법은 시스템의 내부 상태를 시계열 행렬로 표현함으로써 고차 시스템을 1차 미분 방정식의 묶음으로 단순화할 수 있는 장점이 있다.
