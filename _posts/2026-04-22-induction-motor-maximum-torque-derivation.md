## 1. 유도전동기의 토크 정의
유도전동기의 3상 토크 $T$는 2차 입력 $P_2$와 동기 각속도 $\omega_s$를 이용하여 다음과 같이 정의된다.
$$T = \frac{3}{\omega_s} \cdot \frac{V^2}{\left( r_1 + \frac{r_2}{s} \right)^2 + (x_1 + x_2)^2} \cdot \frac{r_2}{s}$$
* $V$: 공급 전압, $s$: 슬립
* $r_1, r_2$: 1, 2차 저항
* $x_1, x_2$: 1, 2차 리액턴스

## 2. 최대토크 발생 조건 유도 (미분)
토크가 최대가 되는 조건은 $\frac{r_2}{s}$에 대해 미분하여 $0$이 되는 지점이다. 계산 편의를 위해 $R = \frac{r_2}{s}$, $X = x_1 + x_2$로 치환하여 몫의 미분법을 적용한다.

$$T = k \cdot \frac{R}{(r_1 + R)^2 + X^2}$$

미분 시 분자가 $0$이 되는 조건:
$$1 \cdot [(r_1 + R)^2 + X^2] - R \cdot [2(r_1 + R)] = 0$$
$$r_1^2 + 2r_1R + R^2 + X^2 - 2r_1R - 2R^2 = 0$$
$$r_1^2 - R^2 + X^2 = 0 \implies R = \sqrt{r_1^2 + X^2}$$

따라서 최대토크 발생 조건 및 그때의 슬립 $s_m$은 다음과 같다.
$$\frac{r_2}{s} = \sqrt{r_1^2 + (x_1 + x_2)^2} \quad \implies \quad s_m = \frac{r_2}{\sqrt{r_1^2 + (x_1 + x_2)^2}}$$

## 3. 최대토크($T_{max}$) 공식 유도
위의 조건을 토크 식에 대입하면 $r_2$ 항이 상쇄되는 과정을 확인할 수 있다. ($A = \sqrt{r_1^2 + X^2}$로 치환)

$$T_{max} = \frac{3 V^2}{\omega_s} \cdot \frac{A}{(r_1 + A)^2 + X^2}$$
분모 정리: $(r_1 + A)^2 + X^2 = r_1^2 + 2r_1A + A^2 + X^2 = 2A^2 + 2r_1A = 2A(A + r_1)$
최종 대입:
$$T_{max} = \frac{3 V^2}{\omega_s} \cdot \frac{A}{2A(r_1 + A)} = \frac{3 V^2}{2\omega_s(r_1 + \sqrt{r_1^2 + (x_1 + x_2)^2})}$$

## 4. 비례추이의 물리적 의미


* **최대 전력 전달 정리:** 유도전동기의 부하 저항 성분($\frac{r_2}{s}$)이 내부 임피던스 크기와 일치할 때 최대 토크가 발생한다.
* **2차 저항($r_2$)의 역할:** $r_2$는 최대토크 공식에서 약분되어 사라지므로, 최대토크의 **크기**에는 영향을 주지 않는다.
* **비례추이:** $r_2$를 증가시키면 $s_m$이 비례하여 증가한다. 즉, 최대토크 값은 일정하게 유지하면서 발생 지점(슬립)만을 이동시키는 것이 비례추이의 핵심이다.
* **간략식:** $r_1$을 무시할 경우 $T_{max} \approx \frac{3 V^2}{2\omega_s(x_1+x_2)}$로 전압의 제곱에 비례하고 리액턴스에 반비례한다.
