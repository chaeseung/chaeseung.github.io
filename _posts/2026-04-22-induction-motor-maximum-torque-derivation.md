## 1. 개요
유도전동기에서 비례추이는 2차 저항($r_2$)을 조절하여 최대토크가 발생하는 슬립($s_m$)을 변화시키는 원리이다. 이때 최대토크의 절대적인 크기는 2차 저항의 값에 영향을 받지 않고 일정하게 유지된다.

## 2. 최대토크 발생 조건 유도
유도전동기의 토크 $T$ 식은 다음과 같다.
$$T = \frac{3}{\omega_s} \cdot \frac{V^2}{\left( r_1 + \frac{r_2}{s} \right)^2 + (x_1 + x_2)^2} \cdot \frac{r_2}{s}$$

토크가 최대가 되기 위해서는 $\frac{r_2}{s}$에 대한 미분값이 0이 되어야 하며, 임피던스 매칭 조건에 의해 다음과 같은 관계가 성립한다.
$$\frac{r_2}{s} = \sqrt{r_1^2 + (x_1 + x_2)^2}$$

따라서 최대토크가 발생하는 슬립 $s_m$은 다음과 같이 $r_2$에 비례한다.
$$s_m = \frac{r_2}{\sqrt{r_1^2 + (x_1 + x_2)^2}}$$

## 3. 최대토크($T_{max}$) 공식 유도
최대토크 조건($\frac{r_2}{s} = A$, 단 $A = \sqrt{r_1^2 + (x_1 + x_2)^2}$)을 원래의 토크 식에 대입하여 정리하면 다음과 같다.

1.  **식 대입:**
    $$T_{max} = \frac{3 V^2}{\omega_s} \cdot \frac{A}{(r_1 + A)^2 + (x_1 + x_2)^2}$$
2.  **분모 전개 및 정리:**
    $$(r_1 + A)^2 + (x_1 + x_2)^2 = r_1^2 + 2r_1A + A^2 + (x_1 + x_2)^2$$
    $A^2 = r_1^2 + (x_1 + x_2)^2$이므로:
    $$= 2A^2 + 2r_1A = 2A(A + r_1)$$
3.  **최종 공식:**
    $$T_{max} = \frac{3 V^2}{2\omega_s(r_1 + \sqrt{r_1^2 + (x_1 + x_2)^2})}$$

## 4. 결론
유도된 $T_{max}$ 공식 내에는 2차 저항($r_2$) 항이 존재하지 않는다. 이는 2차 저항이 변화하더라도 최대토크가 발생하는 슬립 지점만 이동할 뿐, 최대토크의 크기 자체는 변하지 않음을 수학적으로 증명한다. 1차 저항 $r_1$을 무시할 경우 최대토크는 다음과 같이 간략화된다.
$$T_{max} \approx \frac{3 V^2}{2\omega_s(x_1+x_2)}$$
